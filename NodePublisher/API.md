# NodePublisher API

## 属性 options
### streamURL
>完整的流地址

### videoWidth
>视频宽

### videoHeight
>视频高

### videoFPS
>视频帧率

### keyFrameInterval
>关键帧间隔

### bitrate
>视频比特率,单位bit

### videoQuality
>视频清晰度,0-100.

## 方法
### setOptions()
>设置属性,传入options对象

### selectCamera() 
>选择摄像头

### selectMicrophone()
>选择麦克风

### preview()
>开始预览

### start()
>开始发布

### stop()
>停止发布
